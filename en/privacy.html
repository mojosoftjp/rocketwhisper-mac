Privacy Policy

Learn how RocketWhisper for Mac handles your data.

Last updated: February 2, 2026

This Privacy Policy explains how RocketWhisper for Mac (the "App") handles your data. Mojosoft Co., Ltd. ("we", "us", or "our") has designed and developed this App with user privacy as our top priority.

1Core Principle

Completely Offline & Local Processing

All speech recognition processing happens entirely on your Mac. Your voice data and transcription results are never sent to external servers.

2Data We Don't Collect

This App does not collect any of the following:

  • Voice data (recorded audio)
  • Speech recognition results (text data)
  • Personally identifiable information (name, email, IP address, etc.)
  • Usage telemetry data
  • Location information
  • Device identifiers

3Local Processing

The App's speech recognition uses the WhisperKit framework, running locally on Apple Silicon (Neural Engine + Metal GPU).

Processing Flow

  1. Record audio from your microphone
  2. WhisperKit analyzes the audio on your Mac (Neural Engine + Metal GPU)
  3. Recognition results are displayed in the app

Throughout this entire process, no data is transmitted over the internet.

4Data Stored Locally

The App stores only necessary data locally on your Mac.

Settings (UserDefaults)

Selected Whisper model, recognition language, microphone device, hotkey settings, auto copy/paste settings, window position and size

~/Library/Preferences/biz.mojosoft.RocketWhisper.plist

API Keys (Keychain)

API keys for AI providers (encrypted in macOS Keychain)

macOS Keychain (encrypted)

Recognition History (Local File)

Past recognition text, timestamps, source info (microphone/file)

~/Library/Application Support/RocketWhisper/

Word Dictionary & Correction Rules

Registered terms, error correction rules, voice command settings

~/Library/Application Support/RocketWhisper/

WhisperKit Model Files

Downloaded Whisper models (CoreML format)

~/Library/Application Support/RocketWhisper/Models/

How to Delete Data

  1. Uninstall RocketWhisper (delete from Applications folder)
  2. Delete the ~/Library/Application Support/RocketWhisper folder
  3. Delete ~/Library/Preferences/biz.mojosoft.RocketWhisper.plist (optional)

5Third-Party Sharing

Since this App doesn't send data externally, there is no sharing of data with third parties.

6Network Communication

WhisperKit Model Download

Item Details
Destination Hugging Face (model hosting service)
Purpose Downloading speech recognition model files
Data Sent None (download request only)

After downloading the model, the App works completely offline.

AI Processing (Optional)

Connection Type Description
Local LLM
(LM Studio / Ollama)		 No data is sent externally. Fully offline AI processing possible.
Cloud API
(OpenAI / Anthropic / Groq / Gemini)		 Recognition result text is sent to each provider's servers.

Fully Offline Operation

By selecting a local LLM (LM Studio or Ollama), you can use all features including AI formatting completely offline.

7Security

  • All processing happens locally on your Mac
  • Protected by App Sandbox + Hardened Runtime
  • API keys are encrypted in macOS Keychain
  • No network data transmission means no risk of man-in-the-middle attacks

8Children's Privacy

This App has no age restrictions. Children under 13 should use the App under parental supervision. Since the App doesn't collect personal information, privacy risks for children are minimal.

9Policy Changes

We may update this Privacy Policy as needed. Any significant changes will be announced on this page.

Summary

RocketWhisper for Mac fully protects your voice data and privacy. All processing happens on your Mac, with no data ever sent externally. Use with confidence.
