Is internet connection required? +

Speech recognition works completely offline after the initial model download. No internet connection required.

For AI processing features, it depends on the provider:

  • Cloud APIs (OpenAI, Claude, Gemini, Groq) → Internet required
  • Local LLM (LM Studio, Ollama) → Fully offline
Where is my voice data sent? +

Nowhere. All speech recognition processing is completed on your Mac.

When using AI formatting features, only the recognized text (not audio data) is sent to cloud APIs. Using Local LLM keeps everything fully offline.

How accurate is the recognition? +

Accuracy varies by model:

  • Large V3 Turbo — High accuracy & speed (Recommended)
  • Large V3 — Highest accuracy
  • Medium — High accuracy
  • Small — Good accuracy

For Japanese speech recognition, Large V3 Turbo or higher is recommended. Small/Medium may have reduced accuracy for kanji and katakana words.

Registering technical terms in the word dictionary can further improve accuracy.

Does it work on non-Apple Silicon Macs? +

Apple Silicon is recommended, but it works on Intel Macs too. However, processing speed will be slower without Neural Engine. Small or Medium models are recommended for Intel Macs.

Does it support multiple languages? +

Yes. The following languages are supported:

  • Japanese
  • English
  • Chinese
  • Korean
  • Auto language detection
Can I save recognition results? +

Yes. Recognition results can be saved and used in the following ways:

  • Copy to clipboard
  • Auto-save to recognition history
Technical terms aren't recognized +

Register technical terms in the "Word Dictionary" tab in settings. Pre-registering company names, personal names, acronyms, and technical terms that are hard to recognize will significantly improve accuracy.

You can also set up auto-correction rules for misrecognition patterns in the "Correction Rules" tab. Regex is supported.

I want different settings for different apps +

In the "App-Specific" tab in settings, you can configure different processing modes for each app. For example, use punctuation and AI formatting for text editors, and casual mode for chat apps.

Shortcut doesn't work +

Check Accessibility permissions.

System Settings → Privacy & Security → Accessibility - Grant accessibility permission to RocketWhisper.

Restart the app after granting permission for it to take effect.

What macOS version is required? +

macOS 14.0 (Sonoma) or later is required. Apple Silicon Mac (M1/M2/M3/M4) is recommended.
