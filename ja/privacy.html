プライバシーポリシー

RocketWhisper for Mac におけるデータの取り扱いについて説明します。

最終更新日: 2026年2月2日

本プライバシーポリシーは、RocketWhisper for Mac（以下「本アプリ」）におけるデータの取り扱いについて説明します。Mojosoft Co., Ltd.（以下「当社」）は、ユーザーのプライバシー保護を最優先に考え、本アプリを設計・開発しています。

1基本方針

完全オフライン・完全ローカル処理

音声認識処理はすべてお使いのMac内で行われ、音声データや認識結果が外部サーバーに送信されることは一切ありません。

2収集しないデータ

本アプリは以下のデータを一切収集しません。

  • 音声データ（録音された音声）
  • 音声認識結果（テキストデータ）
  • 個人を特定できる情報（氏名、メールアドレス、IPアドレス等）
  • 使用状況のテレメトリデータ
  • 位置情報
  • デバイス識別情報

3ローカル処理について

本アプリの音声認識機能は、WhisperKitフレームワークを使用し、Apple Silicon（Neural Engine + Metal GPU）でローカル実行されます。

処理の流れ

  1. マイクから音声を録音
  2. お使いのMac上でWhisperKitが音声を解析（Neural Engine + Metal GPU）
  3. 認識結果がアプリ内に表示

この処理の全過程において、データがインターネットを経由することはありません。

4ローカルに保存されるデータ

本アプリは動作に必要なデータをお使いのMac内にのみ保存します。

設定情報（UserDefaults）

選択されたWhisperモデル、認識言語設定、マイクデバイス設定、ショートカット設定、自動コピー/貼り付け設定、ウィンドウ位置・サイズ

~/Library/Preferences/biz.mojosoft.RocketWhisper.plist

APIキー（Keychain）

各AIプロバイダのAPIキー（macOS Keychainによる暗号化保存）

macOS Keychain（暗号化保存）

認識履歴（ローカルファイル）

過去の認識結果テキスト、認識日時、ソース情報（マイク/ファイル）

~/Library/Application Support/RocketWhisper/

単語辞書・修正ルール

登録された専門用語、誤認識修正ルール、音声コマンド設定

~/Library/Application Support/RocketWhisper/

WhisperKitモデルファイル

ダウンロードしたWhisperモデル（CoreML形式）

~/Library/Application Support/RocketWhisper/Models/

データの削除方法

  1. RocketWhisperをアンインストール（Applicationフォルダから削除）
  2. ~/Library/Application Support/RocketWhisper フォルダを削除
  3. ~/Library/Preferences/biz.mojosoft.RocketWhisper.plist を削除（任意）

5第三者への提供

本アプリはデータを外部に送信しないため、第三者へのデータ提供は一切行いません。

6ネットワーク通信

WhisperKitモデルのダウンロード

項目 詳細
接続先 Hugging Face（モデルホスティングサービス）
目的 音声認識モデルファイルのダウンロード
送信データ なし（ダウンロードリクエストのみ）

モデルダウンロード後は完全にオフラインで動作します。

AI処理機能（オプション）

接続方式 説明
ローカルLLM
(LM Studio / Ollama)		 データは外部に送信されません。完全オフラインでAI処理可能。
クラウドAPI
(OpenAI / Anthropic / Groq / Gemini)		 認識結果テキストが各プロバイダのサーバーに送信されます。

完全オフライン運用

ローカルLLM（LM Studio または Ollama）を選択すれば、AI整形機能も含めて完全オフラインで利用できます。

7セキュリティ

  • すべての処理がローカルで完結
  • App Sandbox + Hardened Runtime で保護
  • APIキーはmacOS Keychainで暗号化保存
  • ネットワーク経由でのデータ送信がないため中間者攻撃のリスクなし

8お子様のプライバシー

本アプリには年齢制限はありません。13歳未満のお子様は保護者の監督のもとでご利用ください。本アプリは個人情報を収集しないため、お子様のプライバシーに関するリスクは最小限です。

9プライバシーポリシーの変更

当社は必要に応じて本プライバシーポリシーを更新することがあります。重要な変更がある場合は、本ページにて通知いたします。
